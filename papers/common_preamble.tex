% Thematic Transference Principle:
%   Every symbol carries its standard mathematical meaning INTO the
%   new context. "Collisions" are features, not bugs.
%
%   δ = observer AND variation   → algebraic δ-ε limit
%   ε = noise AND tolerance      → ε → 0 IS the limit
%   λ = depth AND wavelength     → λ counts integration cycles
%   Λ = lift AND promotion       → next wavelength
%   Σ = consolidation AND sum    → discrete sum of continuous ε
%   κ = associator AND curvature → non-associative bending
%   ω = thrust AND transfinite   → exponential limit (H₃)
%
% Capital = Operator, Lowercase = State
